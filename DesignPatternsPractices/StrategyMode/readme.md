# 策略模式

标签（空格分隔）： 设计模式

---

编程是一门技术，更是一门艺术。

#### 1、策略模式（Strategy）
它定义了算法家族，分别封装起来，让他们之间可以相互替换，此模式让算法的变化，不会影响到使用算法的客户。
